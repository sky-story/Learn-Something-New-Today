Introduction

Markdown cells are used to add formatted text, headings, lists, links, images, and code snippets alongside your code. This makes your notebook readable and professional.

Where You Can Use Markdown Cells

Google Colab

Jupyter Notebook

(Anaconda, local server)

VS Code

(with Python + Jupyter extensions)

Create or Select a Cell

1

In Colab or Jupyter

Click + Text (Colab) or + to add a new cell

→
2

In VS Code

Add a new cell (+ Code or + Markdown)

Change to Markdown Mode

Platform How to Convert Cell to Markdown
Colab Click the dropdown (left of the cell) and choose "Text"
Jupyter Press Esc, then press M
VS Code Use the #%% [markdown] magic comment above the cell, or select Markdown from the cell type menu

Write Markdown Content

Basics Syntax Output
Heading 1 # Heading 1 Large Heading
Heading 2 ## Heading 2 Medium Heading
Bold **bold** or __bold__ bold
Italic *italic* or _italic_ italic
Bullet List - Item 1
- Item 2		 • Item 1
• Item 2
Numbered List 1. Item
2. Item		 1. Item
2. Item
Code (inline) `code` code
Code Block ```python
print("Hello")
```		 Code block
Link [Google](https://google.com) Google
Image ![alt text](image_url) Image
Horizontal Line --- ___________

Render the Markdown

Platform How to Render
Colab Click outside the cell or press Shift + Enter
Jupyter Press Shift + Enter
VS Code Run the cell (Shift + Enter) — it will render below
