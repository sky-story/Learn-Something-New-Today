Introduction
Python is a high‑level, general‑purpose language whose core idea is “readability first.” It supports procedural, object‑oriented, and functional styles; the reference interpreter (CPython) is open‑source and runs on every major OS. Learning Python well pays off because the same skills transfer to data science, web back‑ends, automation scripts, and more.

Functions

Define with def name(params).
Flexible parameters: positional, keyword‑only (*), default values, *args, **kwargs.
Functions are first‑class objects—store them in variables, pass as arguments, return them.

Lambda: small anonymous function: key=lambda x: x.id.
Decorators: functions that wrap other functions to transparently extend behavior.
